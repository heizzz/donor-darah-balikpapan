# Donor Darah Balikpapan

## Deskripsi Aplikasi
Aplikasi ini dibuat untuk membantu masyarakat untuk membuat janji melakukan donor darah, dan juga untuk melakukan pengecekkan jumlah stok darah pada setiap rumah sakit yang didaftarkan.

## Dokumentasi
Dokumentasi terdapat di dalam folder `Docs`

## Instalasi
1. Clone projek ini
2. Buka folder App
3. Jalankan `composer install`
4. Jalankan `npm install`
5. Jalankan `npm run dev`
6. Copy `.env.example` ke `.env`
7. Jalankan `php artisan key:generate`
8. Jalankan `php artisan migrate`
9. Jalankan `php artisan serve`
10. Buka `localhost:8000`
